Hypr helper scripts

Place these in `~/.config/hypr/scripts/` (they are deployed from `home_config/`).

- `screenshot_selection.sh` — selection capture (uses `slurp` + `grim`, prefers `hyprshot`).
- `screenshot_window.sh` — select and capture a window/area.
- `screenshot_display.sh` — full-display capture.
- `lock.sh` — runs `hyprlock`.

Make scripts executable: `chmod +x ~/.config/hypr/scripts/*.sh`
